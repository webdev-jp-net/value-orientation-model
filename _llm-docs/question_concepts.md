## 2. 各設問のコンセプト詳細

### 帰属志向（valueLocus）

#### [valueLocus-1]

- **質問文**: コンセプトや設計、訓練など外から見えない分野にも関心があり、研究したり手をかけたくなってしまう。
- **評価ラベル**: [max] 熱中してしまう / [min] 内側は最低限で適切
- **コンセプト**:
外から可視化されない内在構造や完成度そのものに価値を感じ、関与が深くなる志向。

#### [valueLocus-2]

- **質問文**: 責任感があり、ついつい関与が深くなって疲れてしまうことがある。
- **評価ラベル**: [max] そうなりがち / [min] 疲れない運用をしている
- **コンセプト**:
責任の所在を自分側に引き寄せやすく、手放さず抱え込むオーナーシップ傾向。

#### [valueLocus-3]

- **質問文**: 結果が喜ばれるとしても、モノの品質を妥協する解決は自分の満足とは別物。
- **評価ラベル**: [max] 妥協した違和感は残る / [min] 喜ばれれば純粋に満足
- **コンセプト**:
外部評価よりも内在的な完成度や納得を判断基準として保持する志向。

#### [valueLocus-4]

- **質問文**: 0→1という、まだこの世にないものの姿を決め形作るフェーズには特別な手応えを感じる。
- **評価ラベル**: [max] まかせて / [min] 得意な人に頼る
- **コンセプト**:
未定義状態から構造や方向性を決める初期フェーズに強い手応えを感じる特性。

#### [valueLocus-5]

- **質問文**: 決定案から自分の関与やアイデアが全てなくなってしまうと、内心ざんねんだ。
- **評価ラベル**: [max] 内心そうだ / [min] 決まれば純粋に満足
- **コンセプト**:
成果よりも関与や思考の痕跡が残ることに価値を感じる内在的満足の傾向。

#### [valueLocus-6]

- **質問文**: どこに出すか、どう伝えるか、どう根回ししておくかで価値を動かすことに関心がある。
- **評価ラベル**: [max] とても意義を感じる / [min] それほど関心がない
- **コンセプト**:
内容そのものよりも流通・配置・合意形成によって価値が変わる点への関心。

#### [valueLocus-7]

- **質問文**: アウトプットの運用に重心を置いているので、 立ち上げや過程のキャッチアップが相対的に浅くなることがある。
- **評価ラベル**: [max] はい、出すときが自分の出番 / [min] 途中の重心のほうが強い
- **コンセプト**:
制作過程よりも公開・展開・運用フェーズを主戦場と捉える志向。

#### [valueLocus-8]

- **質問文**: 反応や評価が返ってきたとき、ようやく一区切りついた気持ちになる。
- **評価ラベル**: [max] 受け入れてもらえるまではアサイン中 / [min] 評価は形が出せたあとの追加要素
- **コンセプト**:
形を出すことより、外部からの反応によって完了感が成立する価値観。

#### [valueLocus-9]

- **質問文**: 好きではないタスクも、必要とされることだと捉えて取り組む。
- **評価ラベル**: [max] 割り切って進めるのは得意 / [min] 分かるけど手は遅い
- **コンセプト**:
嗜好と切り離し、役割や期待に応じて行動を選択できる運用志向。

#### [valueLocus-10]

- **質問文**: 自分の勉強を共有する場所があると、意味や納得の裏付けになり安心感がある。
- **評価ラベル**: [max] 外からも認められると心強い / [min] 意味は自分で決める
- **コンセプト**:
内省だけでなく、外部からの承認や共鳴によって納得感が補強される特性。

### 関係性志向（boundary）

#### [boundary-1]

- **質問文**: 「話題だから美味しい」「有名だからおしゃれ」という評価のしかたは違和感がある。
- **評価ラベル**: [max] 違和感を感じる / [min] どこに違和感？
- **コンセプト**:
話題性や権威と評価軸を混線させることへの直感的な境界検知。

#### [boundary-2]

- **質問文**: さまざまな価値観が交錯する物語を、どの登場人物や勢力にも自己投影せず楽しめる。
- **評価ラベル**: [max] 興味深い世界観だ / [min] 誰かを応援する派
- **コンセプト**:
正義や立場を固定せず、複数の解釈を並列で保持できる境界特性。

#### [boundary-3]

- **質問文**: 自分がおすすめしたものへ相手は興味がなくてもかまわない。
- **評価ラベル**: [max] ぜんぜん平気 / [min] ややさみしい
- **コンセプト**:
自分の評価と他者の関心を分離して扱える境界の安定性。

#### [boundary-4]

- **質問文**: 状況に応じた立ち回りが求められる交流会や懇親会では、消耗を感じやすい。
- **評価ラベル**: [max] 消耗する / [min] ぜんぜん自然体でいける
- **コンセプト**:
文脈適応や関係調整を同時処理する場面での境界負荷の検知。

#### [boundary-5]

- **質問文**: いつもキレっキレの提案をする人がパッとしなくても、評価や信頼感は大きく揺らがない。
- **評価ラベル**: [max] 今回は調子悪いね / [min] ギャップに内心ざわつく
- **コンセプト**:
能力評価を一時的な成果変動と切り分けて捉えられる境界感覚。

#### [boundary-6]

- **質問文**: 物事を進行するとき、人間関係でまとめる立ち回りが自分には合っている。
- **評価ラベル**: [max] 気がつけばそこに寄ってる / [min] よくわからない、自分はしない。
- **コンセプト**:
関係性を軸に進行を安定させる融合的な進め方への親和性。

#### [boundary-7]

- **質問文**: 「Aさん絶対すごいよね！」と切り出されると、まず「すごいよね」と同意する。
- **評価ラベル**: [max] それ、自分のことだ / [min] まずは「ふーん」
- **コンセプト**:
他者の評価フレームに即座に同調し、自身の判断と混線させやすい傾向。

#### [boundary-8]

- **質問文**: レビューやおすすめを、共感したまま自分の評価として表明していることがある。
- **評価ラベル**: [max] いつのまにかやりがち / [min] おすすめは参考止まり
- **コンセプト**:
他者の感想を自分の評価として内面化・表明する境界の曖昧さ。

#### [boundary-9]

- **質問文**: グループでの取り組みは、推進委員や代表がリードする体制がわかりやすくて安心。
- **評価ラベル**: [max] そうだと思う / [min] 場合による
- **コンセプト**:
代表への委譲や一体感によって安心を得る融合志向。

#### [boundary-10]

- **質問文**: 多数決で決めきるとき、賛成ではない人が残ることを内心では割り切りにくい。
- **評価ラベル**: [max] 仕方ないけど気がかり / [min] 決まればどっちでもいい
- **コンセプト**:
決定結果よりも関係の残余や同質性を気にかける融合的感受性。
